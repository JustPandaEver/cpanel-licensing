# Cpanel Licensing
This project  is about installing cpanel and wordpress toolkit license using golang on CentOS system.

## Instructions

- Install with Go
```sh
git clone https://github.com/faithcredit/cpanel-licensing.git
cd cpanel-licensing
go run main.go
```
